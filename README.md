# NEM12 Parser

A Go-based parser for NEM12 energy meter data files with database inspection capabilities.

## 📋 Overview

This project provides tools to parse NEM12 (Australian National Electricity Market) data files and inspect the resulting database.

## 🚀 Quick Start

### Running the Parser

```bash
go run . data/sample.NEM12
```

### Running Tests

```bash
go test ./...
```

### Database Inspection

```bash
python query.py
```

## 📚 Documentation

Detailed write-ups and additional documentation are available in the `docs/` folder.

## 📁 Project Structure

- `docs/` - Documentation and write-ups
- `data/` - Sample NEM12 data files

## 🛠️ Technologies

- **Language**: Go
- **Database Inspection**: Python
